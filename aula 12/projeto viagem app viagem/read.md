Startup: App de Intercâmbio Inteligente 
1. Problema e Dados 
Muitos estudantes têm dificuldade em organizar informações 
sobre intercâmbio (custos, documentos, moradia e trabalho). 
Dados brutos coletados: 
• Orçamento do usuário  
• Nível de inglês  
• País de interesse  
• Custos de passagem, moradia e alimentação  
• Requisitos de visto  
• Vagas de emprego no exterior 
lassificação dos dados: 
• Entrada (input): dados inseridos pelo usuário (perfil, 
dinheiro disponível, objetivo)  
• Origem dos dados: 
o usuário  
o sites de intercâmbio  
o bases de dados públicas (custos de vida, vistos)  
• Tipo de dados: 
o estruturados: valores, datas, países  
o não estruturados: descrições, preferências, 
comentários 
2. Processamento 
O sistema transforma dados em informação através das 
seguintes etapas: 
• Coleta: dados do usuário e de fontes externas  
• Armazenamento: banco de dados na nuvem  
• Processamento: organização e cruzamento dos dados  
• Análise: uso de algoritmos e inteligência artificial para gerar 
recomendações  
Tecnologias utilizadas: 
• banco de dados (ex: MySQL)  
• algoritmos de recomendação  
• inteligência artificial  
• dashboards para visualização  
Relação com o sistema: 
• Sistema: aplicativo de intercâmbio  
• Processos: coleta, análise e recomendação  
• Banco de dados: armazenamento das informações  
• Fluxo da informação: usuário → sistema → análise → 
resultado 
O sistema gera: 
• Padrões: países mais escolhidos por perfil  
• Tendências: destinos mais acessíveis financeiramente  
• Previsões: custo total do intercâmbio e chances de 
sucesso  
Transformação: 
• Dados → custo, idioma, documentos  
• Informação → custo médio por país  
• Conhecimento → melhor escolha para o usuário  
Exemplo: 
“O usuário tem perfil ideal para estudar e trabalhar na Irlanda 
com orçamento X. 
Decisão e Valor Permite decisões como escolha do país, 
planejamento financeiro e organização da viagem. Valor: 
eficiência, redução de custos, personalização e inovação.
